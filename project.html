Overview

This is my Tower Defense game, currently under development. The goal of this game is to build towers and defend against waves of enemies. The twist, you might ask? This game will have a playable hero character that lets you further interact with the towers' enemies and obstacles the hope is to almost mix tower defence and rogue like into an interesting and replayable game.

Technologies Used

  • Unity
  • C#
  • Tilemap System

Project Challenges

Throughout the development of this game, I faced challenges related to performance optimization and balancing the game mechanics. I worked through these by using object pooling techniques and adjusting difficulty curves.

Future Plans

I plan to add more levels, improve AI pathfinding, and integrate a scoring system in future updates. There's also potential for multiplayer features down the line.

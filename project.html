Overview

This is my Tower Defense game, currently under development. The goal of this game is to build towers and defend against waves of enemies. The twist, you might ask? This game will have a playable hero character that lets you further interact with the towers' enemies and obstacles the hope is to almost mix tower defense and rogue-like into an interesting and replayable game.

Technologies Used

  • Unity
  • C#
  • Tilemap System
